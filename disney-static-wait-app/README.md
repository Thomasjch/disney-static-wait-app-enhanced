# Disney Toddler Wait Navigator

Static no-build app. Upload `index.html` to Vercel, Netlify, GitHub Pages, or any static host.

Features:
- Live Queue-Times wait data
- Auto-refresh every 60 seconds
- Map and ranked list
- Best Next recommendations
- All four Walt Disney World parks
- Toddler / 40-inch filter
- Indoor, character, and Lightning Lane filters
- Optional phone location for walking estimates

Vercel settings:
- Framework Preset: Other
- Build Command: blank
- Output Directory: blank or `.`
